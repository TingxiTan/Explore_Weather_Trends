# Explore_Weather_Trends
*author: Tingxi Tan*

This is a draft of my submission for Udacity Data Analyst Nanodegree program. In this project we were given the task of analyzing local and global temperature data and comparing the temperature trend between our residence with global temperature trend. I used SQL queries to extract data, creat a visualization, and prepared a document to describe the similarities and differences of global temperature trends between with our residence's temperature.
